\caption{Scattering of a plane wave by a fictitious interface with a  bump, glass-to-air. The lower region ($\eps_1=1$) is the same material
         as the upper region air ($\eps_2=1$). The wave should propagate through the interface with no changes.
         The wave number of the incident field is $k_z=5$. Results from 
         CGFD4. Left: results using grid $\Gc^{(4)}$ (6M grid points, $\ppw=16$). Right: results from $\Gc^{(8)}$ (41M grid points, $\ppw=32$). The
         intensity is close to $1$ over the entire domain. There are some reflections from the top far-field boundary which
