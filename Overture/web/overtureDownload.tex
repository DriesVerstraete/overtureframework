Here is where you can download {\bf Overture}, {\bf A++/P++} and the {\bf cg} PDE solvers.
Note that Overture will only run on Unix or Linux machines since we use Motif for the
graphical user interface. If you really want to run on Windows you could try using the cygwin
unix port.
% (see \htmladdnormallink{http://slyi.ath.cx/projects/overture}{http://slyi.ath.cx/projects/overture}).

Here are \htmladdnormallink{installation instructions}{../install/index.html} for Overture.

\noindent{\bf HINT:} Keep a link to this page if you want to avoid registering again for Overture.

class library for serial and parallel arrays. This version should compile with newer
versions of the C++ compilers.

\noindent
\htmladdnormallink{cg.v24.tar.gz}{\software/cg.v24.tar.gz} : The {\bf cg} PDE solvers. 
Requires the same version of Overture to be installed.
